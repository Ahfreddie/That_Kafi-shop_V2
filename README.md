☕ Kafi Shop Landing Page (V2)
📌 Overview

Kafi Shop is a Scandinavian-themed coffee shop landing page redesigned from a previous version. The project focuses on improving layout structure, styling, and overall user experience.

It draws inspiration from the Starbucks landing page while applying custom colors and design adjustments to create a unique look.

🎯 Objective
Strengthen HTML and CSS skills
Practice responsive design and layout structuring
Improve a previous project with better design decisions
Explore clean, minimal Scandinavian design
🧠 Key Focus Areas
Semantic HTML
CSS (Flexbox/Grid)
Responsive design
Layout and spacing
Component-based structure
🎨 Design Style

Minimal, clean, and functional—following Scandinavian design principles with soft colors and simple layouts.
